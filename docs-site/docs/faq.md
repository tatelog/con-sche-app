# よくある質問

## 登録について

### 本当に無料ですか？

はい。エディタの全機能を無料で利用できます。連携API（CPM計算・ファイル変換）も月間無料枠の範囲で無料で試せます。無料枠を超える本格的なシステム連携・組み込み利用は個別契約にて承ります。

### フリーメールで登録できますか？

いいえ。法人メールアドレスのみ登録可能です。Gmail、Yahoo!メール等のフリーメール・携帯キャリアのアドレスは使用できません。

### パスワードはありますか？

ありません。アカウント制ではなく、初回のみ氏名・会社名・メールアドレスを登録する方式です。

### APIコードを紛失しました

APIコードは再発行できません。エディタの利用には影響ありませんが、API連携を利用したい場合は[お問い合わせ](https://con-sche.tatelog.biz/#contact)ください。

## 工程表データについて

### データはどこに保存されますか？

お使いの端末のブラウザ内（IndexedDB）に自動保存されます。**サーバーには送信されません。** ブラウザのキャッシュクリアやシークレットモードではデータが消失する可能性があるため、大事な工程表は .csa ファイルとして書き出して保管してください。

### 別のブラウザ・デバイスからアクセスできますか？

データはブラウザ固有のストレージに保存されるため、そのままでは他の端末から見えません。.csa ファイルに書き出して、別の端末で読み込んでください。

## 操作について

### 歩掛計算はどう使いますか？

アクティビティのプロパティパネルで「工期計算モード」を「歩掛計算」に切り替えると、数量・人数・歩掛から自動的に作業日数が算出されます。歩掛マスタからの選択も可能です。

### 進捗線（雷線）はどう追加しますか？

ツールバーの進捗線ボタン（⚡）で進捗線モードに切り替え、基準日を設定してから各行の進捗位置をクリックします。

### IFCファイルをインポートできますか？

現在準備中です（近日対応予定）。基本的な工程表作成にBIMデータは不要です。

## エクスポートについて

### どの形式でエクスポートできますか？

- **PDF** — A1〜A4サイズ（表題欄・押印欄カスタマイズ可能）
- **CSV** — アクティビティ一覧・CPM結果を含む
- **Con-Scheファイル** — フルパッケージ（.csa）/ ライト（.csl）

### 表題欄はカスタマイズできますか？

はい。PDF出力ダイアログで、表題欄の内容（現場名・工事名等）や押印欄の設定を変更できます。

## 連携APIについて

### どんなことができますか？

外部システムやAIエージェントから、CPM計算（クリティカルパス・フロート・工期算出）と工程ファイル変換（JSON ⇔ .csa）を呼び出せます。詳細は[APIドキュメント](https://con-sche.tatelog.biz/api-docs)を参照してください。

### 料金はかかりますか？

月90ポイントの無料枠があります（読み取り系2pt/リクエスト、成功時のみ消費）。無料枠を超える利用は個別契約にて承ります。
