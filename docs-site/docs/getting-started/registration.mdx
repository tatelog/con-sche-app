# 利用者登録

Con-Scheは無料で利用できます。どなたが利用されているかを把握するため、**初回のみ**利用者登録をお願いしています。アカウントやパスワードはありません。

## 登録手順

1. [Con-Scheトップページ](https://con-sche.tatelog.biz)の **「無料で使う」** をクリック
2. エディタを開くと登録フォームが表示されるので、以下を入力：
   - **氏名**
   - **会社名**
   - **メールアドレス**（会社ドメイン必須）
3. **「登録して無料で使う」** をクリック

:::caution
フリーメール（Gmail、Yahoo!メール等）・携帯キャリアのメールアドレスでは登録できません。会社のメールアドレスをご使用ください。
:::

## APIコードについて

登録が完了すると、**APIコード（`cs_live_...`）が1回だけ表示されます**。

- APIコードは外部システム連携・AI連携（[連携API](https://con-sche.tatelog.biz/api-docs)）で使用します
- **エディタを使うだけなら不要**です
- **再発行はできません。** 画面を閉じると二度と表示されないため、API連携を使う予定がある方は必ずその場で控えて安全な場所に保存してください

## 登録後

登録はブラウザごとに記録されます。別のブラウザ・端末で使う場合は、同じ情報で再度登録してください（登録済みメールアドレスの場合、APIコードは再発行されずそのまま利用開始できます）。
