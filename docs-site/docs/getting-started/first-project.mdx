# 最初の工程表作成

## 工程表データの保存場所

工程表はお使いの端末のブラウザ内（IndexedDB）に保存されます。サーバーには送信されません。端末をまたいで使う場合や長期保管には、[.csaファイルへの書き出し](../network-schedule/export)を利用してください。

## 新しい工程表を作る

1. エディタ（`/app`）を開くと、空の工程表が表示されます
2. ツールバーの **⚙️ 設定** からプロジェクト名・着工日・表示モードなどを設定します
3. **描画モード**でキャンバスをクリックし、工程を描き始めます（[アクティビティの追加](../network-schedule/add-activities)）

編集内容は自動保存されます。ツールバーのプロジェクト名の横に「自動保存済 HH:MM」と表示されます。

## 保存・複製

ツールバーの **工程表メニュー（📁）** から操作します：

| メニュー項目 | 説明 |
|-------------|------|
| 上書き保存 | 現在の工程表をブラウザ内に保存 |
| 別名で保存... | 別名でコピーを保存（複製として使えます） |
| ファイルを開く... | .csa / .csl ファイルを読み込み |

## 保存済み工程表を開く

1. ツールバー左端の **← 工程表一覧** をクリック
2. ブラウザ内に保存された工程表の一覧が表示されます
3. **開く** で読み込み、**削除** で削除できます（削除は取り消せません）

:::caution バックアップのすすめ
ブラウザのデータ消去（キャッシュクリア等）で工程表が消える可能性があります。大事な工程表は **.csaファイルとして書き出して**保管してください。
:::
