# 基本操作

## エディタの構成

エディタは以下の要素で構成されています：

- **ツールバー**（上部） — 各種操作ボタン
- **左ヘッダーパネル**（左側） — 工区・階数等の分類
- **カレンダーヘッダー**（上部） — 日付表示
- **キャンバス**（中央） — 工程表の描画エリア
- **プロパティパネル**（右側） — 選択した要素の編集

## ツールバー

### ナビゲーション

| ボタン | 説明 |
|--------|------|
| **← 工程表一覧** | ブラウザ内に保存された工程表の一覧を開く |
| **プロジェクト名** | クリックでインライン編集可能。右に自動保存ステータス表示 |

### 工程表メニュー（📁 ドロップダウン）

| メニュー項目 | 説明 |
|-------------|------|
| 上書き保存 | 現在の工程表を保存 |
| 別名で保存... | 別名でコピーを保存 |
| ファイルを開く... | .csa / .csl ファイルを読み込み |
| IFCインポート（準備中） | 現在準備中の機能です |

### 印刷・出力メニュー（🖨️ ドロップダウン）

| メニュー項目 | 説明 |
|-------------|------|
| 印刷 / PDF出力... | PDF出力ダイアログを開く |
| CSV出力 | アクティビティデータをCSVでダウンロード |

### 編集ツール

| ボタン | 説明 |
|--------|------|
| ⚙️ 設定 | プロジェクト設定ダイアログを開く |
| 🖱️ 選択モード | **選択** — 要素をクリック・ドラッグで選択 / **パス選択** — アクティビティの経路を選択 |
| ✏️ 描画モード | **描画** — ノード・アクティビティを作成 / **テキスト** — テキストボックスを配置 |
| ⚡ 進捗線 | 進捗線（雷線）モードに切り替え |
| 📈 Sカーブ | Sカーブ（進捗曲線）の表示/非表示を切り替え |
| ↩️ 元に戻す | 直前の操作を取り消し（Ctrl+Z） |
| ↪️ やり直し | 取り消した操作をやり直し（Ctrl+Y） |

### 表示設定

| ボタン | 説明 |
|--------|------|
| 🔍- 縮小 | キャンバスを縮小 |
| **100%** | クリックでズームをリセット、現在の倍率を表示 |
| 🔍+ 拡大 | キャンバスを拡大 |
| 行数 | 表示行数を変更 |
| 行高 | 行の高さをスライダーで調整 |

## キャンバス操作

| 操作 | 動作 |
|------|------|
| マウスホイール | 上下スクロール |
| Shift + ホイール | 左右スクロール |
| Ctrl + ホイール / Ctrl + ;/- | ズームイン・アウト |
| Ctrl + 0 | ズームリセット |
| スクロールバー | キャンバス右端・下端のバーをドラッグ |
| Space | 描画⇔進捗線モードの切替 |
| 右クリック | 連続描画の終了 / テキストモード解除 |
| Shift + 右クリック | コンテキストメニュー |
| Delete | 選択要素を削除 |
| Esc | 選択解除 |

タブレット・スマホではピンチでズーム、ドラッグでスクロールできます。
