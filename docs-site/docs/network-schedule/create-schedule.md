# 工程表の新規作成

## 作成手順

1. エディタ（`/app`）を開くと空の工程表が表示されます
2. ツールバーの **⚙️ 設定** からプロジェクト名・着工日を設定します
3. 描画モードで工程を描き始めます

既存の工程表から始める場合は、ツールバーの工程表メニューから **「ファイルを開く...」** で .csa / .csl ファイルを読み込むか、**← 工程表一覧** からブラウザ内に保存済みの工程表を開きます。

## プロジェクト設定

エディタの設定メニュー（⚙️）から変更できます：

| 設定項目 | 説明 |
|----------|------|
| プロジェクト名 | 工程表の名称 |
| 着工日 | 工程の起点となる日付 |
| 表示モード | 週間(2週) / 週間(3週) / マスター |
| 表示日数・行数 | キャンバスに表示する範囲 |
| 行の高さ | ツールバーのスライダーでも調整可能 |

## 左ヘッダー（階層列）の設定

左ヘッダーパネルには複数の階層列を設定できます：

- **工区** — 工事区分
- **階数** — 階数
- **部屋名** — 詳細区分

各列のマスターデータを登録して使い回せます。
