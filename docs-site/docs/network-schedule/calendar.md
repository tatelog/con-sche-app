# カレンダー管理

工程表のカレンダー（稼働日・休日・現場イベント）を管理する方法を説明します。

## 稼働日の設定

デフォルトでは月曜〜金曜が稼働日に設定されています。プロジェクト設定から週の稼働日を変更できます。

| 曜日 | デフォルト |
|------|----------|
| 月〜金 | 稼働日 |
| 土・日 | 休日 |

## 祝日の管理

Con-Scheには日本の祝日（元日、成人の日、建国記念の日、天皇誕生日、春分の日、昭和の日、憲法記念日、みどりの日、こどもの日、海の日、山の日、敬老の日、スポーツの日、文化の日、勤労感謝の日）があらかじめ登録されています。

### 休日の種類

| 種類 | 説明 |
|------|------|
| **祝日** (`national`) | 国民の祝日（プリセット済み） |
| **会社休日** (`company`) | 会社独自の休日（夏季休暇、年末年始等） |
| **カスタム** (`custom`) | ユーザー定義の休日 |

### 稼働日への上書き

祝日や休日を稼働日に変更することもできます。例えば、土曜出勤日を「稼働日」としてマークできます。

## 現場イベント

カレンダーに現場イベントを登録できます。イベントはカレンダーヘッダーに表示されます。

| イベント種類 | 説明 |
|------------|------|
| **検査** (`inspection`) | 各種検査日 |
| **搬入** (`delivery`) | 資材搬入日 |
| **式典** (`ceremony`) | 上棟式、竣工式等 |
| **会議** (`meeting`) | 定例会議等 |
| **その他** (`other`) | 上記以外のイベント |

各イベントには名称、備考、色を設定できます。

## カレンダーと工期計算

カレンダー設定は工期計算に自動的に反映されます：

- 休日はアクティビティの所要日数にカウントされません
- キャンバス上で非稼働日はグレーのハッチングで表示されます
- CPM計算結果の日付もカレンダーを考慮して算出されます
