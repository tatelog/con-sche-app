# エクスポート

作成した工程表をエクスポートする方法を説明します。

## PDF エクスポート（印刷）

メニューから「印刷 / PDF出力」を選択します。

![PDFエクスポート設定画面](/img/pdf-export.png)

### 用紙設定

| 設定 | 選択肢 |
|------|--------|
| 用紙サイズ | A4 / A3 / A2 / A1 |
| 向き | 縦 / 横 |
| 余白 | 上下左右の余白設定 |

### 表題欄

PDFには建設業の慣例に合わせた表題欄が付きます：

- **現場名** — プロジェクト設定の現場名
- **工期** — 着工日〜竣工日
- **作成日** — 出力日
- **改訂番号** — 版数管理
- **情報チェック** — 計画/実施/変更のチェック欄
- **押印欄** — カスタマイズ可能な印鑑スロット
- **フッターテキスト** — 任意のテキスト

複数ページにまたがる場合は、自動的にページ分割されます。

## CSV エクスポート

アクティビティの一覧をCSV形式でエクスポートします。以下の情報が含まれます：

- アクティビティ名・所要日数
- 開始ノード・終了ノード
- CPM計算結果（ES/EF/LS/LF/TF/FF）
- クリティカルパスかどうか

Excel等の表計算ソフトで開いて分析できます。

## Con-Sche独自ファイル形式

Con-Sche独自のZIPベースファイル形式でエクスポートできます。JSON形式と異なり、データ構造が保護された安全な形式です。

### フルパッケージ（.csa）

プロジェクトの全データを含むファイルです。バックアップや別環境への完全移行に最適です。

| 含まれるデータ | 説明 |
|--------------|------|
| プロジェクト設定 | 現場名・工期・表示設定など |
| 工程表データ | ノード・アクティビティ |
| カレンダー | 稼働日・祝日 |
| 階層データ | 工区・部屋・細目 |
| マスタデータ | 各種マスタ・カスタム列 |
| テキストボックス | キャンバス上のテキスト |

### ライト（.csl）

工程とカレンダーのみを含む軽量ファイルです。マスタや階層データは含まれません。工程表の共有に便利です。

| 含まれるデータ | 説明 |
|--------------|------|
| プロジェクト設定 | 現場名・工期・表示設定など |
| 工程表データ | ノード・アクティビティ |
| カレンダー | 稼働日・祝日 |
| テキストボックス | キャンバス上のテキスト |

### インポート

.csa / .csl ファイルを読み込むと、manifest情報からフルパッケージかライトかを自動判別し、対応するデータを復元します。

## SVG プレビュー（実験的機能）

高品質なベクター形式でのプレビューが利用できます。現在は実験的機能として提供されています。
