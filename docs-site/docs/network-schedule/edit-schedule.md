# 工程表の編集・修正

## 編集モード

### 選択モード

ツールバーの選択モードドロップダウンから切り替えます：

- **選択** — 要素をクリック・ドラッグで選択。プロパティパネルで詳細を編集
- **パス選択** — アクティビティの経路（パス）全体を選択し、一括移動が可能

#### パス選択モードの詳細

パス選択モードでは、アクティビティをクリックするとそのパス（経路）全体が選択されます。選択したパスをドラッグすると、**フロート（余裕日数）の範囲内**でパス全体を前後に移動できます。

- **通常パス** — トータルフロート（TF）の範囲内で右方向に移動可能。先行パスとの間隔の範囲で左方向にも移動可能
- **クリティカルパス** — クリティカルパス（TF=0）も移動可能。工期延長を伴う右方向への移動に制限なし

:::tip パス移動の活用
山積み・山崩し（リソース平準化）の検討時に、フロートを活用してパスを移動させることで、人員の集中を避けられます。
:::

### 描画モード

ツールバーの描画モードドロップダウンから切り替えます：

- **描画** — イベントノードとアクティビティを作成
- **テキスト** — テキストボックスを配置（横書き / 縦書き対応）
- **進捗線（雷線）** — 進捗状況を示す雷線を描画

## 要素の編集

### ノードの移動

選択モードでノードをドラッグして移動できます。接続されたアクティビティも自動的に追従します。

### アクティビティの経路変更

アクティビティの経路はウェイポイントで調整できます。プロパティパネルで経路の詳細設定が可能です。

### 要素の削除

要素を選択して **Delete** キーで削除できます。

## テキストボックス

テキストモードでキャンバス上をクリックすると、テキストボックスが作成されます。

- **横書き / 縦書き** — プロパティパネルで切り替え
- **文字色・枠線色・背景色** — カラーピッカーで設定
- **スナップ位置** — 3x3のグリッドで配置位置を指定

## 元に戻す・やり直し

| 操作 | ショートカット |
|------|--------------|
| 元に戻す | Ctrl + Z |
| やり直し | Ctrl + Y |

ツールバーのボタンからも操作可能です。操作履歴がない場合、ボタンはグレーアウトされます。

## 自動保存

工程表は編集中にブラウザ内へ自動保存されます。ツールバーのプロジェクト名の横に保存ステータスが表示されます：

- **「保存中...」** — 保存処理中
- **「自動保存済 HH:MM」** — 最後に保存された時刻

長期保管や端末間の受け渡しには、[.csaファイルへの書き出し](./export)を利用してください。
