PROBLEM

「この仕上げ、なぜ5日なの？」
と聞かれて答えられない

工程表の日数に根拠がない。前の現場の感覚で書いているだけ。

SOLUTION

歩掛×数量で、日数を自動算出

数量と歩掛を入力すれば所要日数を計算。「歩掛から算出: 67日」と答えられる。

PROBLEM

「鉄骨が3日遅れたら
全体はどうなる？」がわからない

バーチャートでは作業の依存関係が見えない。遅延の影響が読めない。

SOLUTION

ネットワーク工程表で
クリティカルパスを自動計算

依存関係を矢印で表現。遅延がどこまで響くか一目瞭然。

工程表が事務所のPCにしかない
